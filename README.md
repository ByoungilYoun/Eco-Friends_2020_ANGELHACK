# 2020 ANGELHACK Seoul 
- 2020 엔젤핵 서울 온라인으로 개최된 대회를 참가했습니다. 

## 소개 
- 팀명 : Eco-Friends.  
- 팀원 : iOS (윤병일, 안준영, 변윤나) , UI/UX (강지현, 이지민).  
- 도전과제 : 우아한 형제들 - 지구를 위한 책임감 있는 소비와 생산을 위한 솔루션.
- 아이디어 및 앱 : 쓸애기 (쓸만한 애기) - 생활 폐기물 수거 서비스.  

## Requirements 
- **Language** 
  - Swift 
- **Framework** 
  - UIKit
  - SPM(SwiftPackageMAnager)
